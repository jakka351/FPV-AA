# FPV Android Auto
The classic Ford Performance Vehicles gauges recreated for Android Auto...

Simply install the apk file to your phone, pair an OBDX Pro FT device to your phone and connect it to the OBD2 port of the vehicle, and then run Android Auto. https://obdxpro.com/ Thanks to JasonACT for extracting the images from the firmware of the FDIM.
